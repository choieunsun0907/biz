배경을 선택하세요

스튜디오풍 배경 5종 중 선택하거나, 직접 촬영한 배경 사진을 레퍼런스로 등록해서 쓸 수 있어요.

로고 (선택, 여러 개 등록 후 하나 선택)
배경과는 별개예요. 여러 개 등록해두고 클릭해서 쓸 로고를 고르면, 썸네일 오른쪽 아래와 상세페이지 맨 아래 클로징에 작게 워터마크로 들어가요. 투명 배경 PNG를 추천해요.

제품 이미지를 업로드하고 분석하세요

한 제품의 사진을 여러 장 올릴 수 있어요. 대표 이미지는 상세페이지의 히어로 컷에 쓰이고, 나머지는 업로드한 순서대로 ②③④⑤... 섹션으로 상세페이지에 자동으로 이어져요. 제품명·코드를 입력하고 'AI 분석 미리보기'까지 이 화면에서 한 번에 끝낼 수 있어요.

클릭하거나 이미지를 끌어다 놓으세요
JPG, PNG 권장 · 여러 장 선택 가능 · 원하는 등장 순서대로 올려주세요
사진을 클릭하면 대표(히어로 컷)로 지정돼요. 나머지는 올린 순서 그대로 ②③④⑤ 섹션이 되고, 사진을 드래그하거나 아래 ‹ › 버튼으로 순서를 바꿀 수 있어요.
제품명·코드를 입력해두면 아래 'AI 분석 미리보기'에서 사진 추측 대신 이 정보를 우선해서 분석하고, 필요하면 검색까지 활용해요.
⚠️ 제품명·코드는 입력 안 해도 되지만, 이걸 입력하지 않은 경우 아래 'AI 분석 미리보기'를 눌러야만 상세페이지에 텍스트가 들어가요. 텍스트를 직접 입력하는 화면은 따로 없어요.

    완성됐어요

    썸네일은 업로드하신 제품 사진을 1단계 배경 위에 즉시 합성해서 바로 보여드려요 (AI 호출 없음). 상세페이지는 아래 "이미지 생성하기" 버튼을 누르면 AI로 자연스럽게 합성해요 — 추가 사진이 있으면 1단계 배경 대신 AI가 사진마다 어울리는 컨셉을 직접 정해요 (사진이 많을수록 시간이 더 걸려요).

    위 아래
