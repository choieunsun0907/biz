이 도구는 Google의 Gemini 2.5 Flash Image (Nano Banana)로 평컷을 마네킹 입체로 변환합니다. 이미지 생성은 Google 결제(billing) 활성화가 필요해요. 신규 가입 시 $300 무료 크레딧이 주어지고, 이미지당 약 $0.04로 거의 무료입니다.

  1. Google AI Studio에서 API 키 발급 (Google 계정 필요)
  2. 같은 화면에서 결제(Billing) 활성화 — 이미지 생성에 필수 (신규 $300 크레딧)
  3. 키를 아래에 붙여넣으면 이 브라우저에 저장됩니다
  4. 키는 외부로 전송되지 않고 Gemini 호출에만 직접 사용돼요

평컷을 올리면, 마네킹 입체 컷의 투명 PNG로 변환합니다.
