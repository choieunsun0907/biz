Gemini API 키 설정

이 도구는 Google의 Gemini 3 Pro Image (Nano Banana Pro)로 평컷을 마네킹 입체로 변환하고 2K(2048px) 고해상도로 출력합니다. 이미지 생성은 Google 결제(billing) 활성화가 필요해요. 신규 가입 시 $300 무료 크레딧이 주어지고, 2K 이미지당 약 $0.13로 저렴합니다.

  1. Google AI Studio에서 API 키 발급 (Google 계정 필요)
  2. 같은 화면에서 결제(Billing) 활성화 — 이미지 생성에 필수 (신규 $300 크레딧)
  3. 키를 아래에 붙여넣으면 이 브라우저에 저장됩니다
  4. 키는 외부로 전송되지 않고 Gemini 호출에만 직접 사용돼요

평컷을 올리면, 마네킹 입체 컷의 투명 PNG로 변환합니다.

앞·뒤를 함께 올리면 앞면·뒷면 결과를 각각 생성합니다. 배경은 자동으로 깨끗이 투명 처리되고, 출력은 2K(2048px) 고해상도 투명 PNG입니다.
